# Automatic Timetable Generator

This is a web application that can generates automatic timetables for various educational institutions, such as schools and colleges.

## Features

- Automatically creates timetables based on course schedules, teacher availability, and classroom assignments.
- Allows administrators to easily manage course information, teacher details, and classroom resources.
- Provides a user-friendly interface for students and teachers to view their personalized timetables.
- Supports customization and integration with existing school management systems.

## Getting Started

1. Clone the repository: git clone https://github.com/KatumbieGeorges/Automatic-Timetable-Generator.git
2. Install the required dependencies: composer install
3. Set up the database connection in the .env file.
4. Run the database migrations: php artisan migrate
5. Start the development server: php artisan serve

## Usage

1. Log in as an administrator to manage course, teacher, and classroom data.
2. Configure the timetable settings based on your institution's requirements.
3. Generate the automatic timetable and review the results.
4. Publish the timetable for students and teachers to access.
